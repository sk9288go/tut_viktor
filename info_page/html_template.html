정림건축 일사분석 Web App

이 앱은 Ladybug Tool과 Grasshopper 스크립트를 활용하여 사용자가 간단한 건축 3D 모델, 매스 또는 파사드를 업로드하고,
이를 통해 일사량을 분석한 후 보고서 형태로 결과를 출력하는 기능을 제공합니다. 사용자는 10MB 이내의 OBJ 파일을 업로드할 수 있습니다.

세부 조정이 필요한 사용자는 Grasshopper 스크립트를 다운로드하여 Ladybug Tool을 설치 후 사용할 수 있습니다.
추가 지원이 필요한 경우, NewTech의 GT팀 또는 DT팀에 문의하시기 바랍니다.

분석을 시작하려면, '분석 요소 설정'으로 넘어가서 OBJ 파일을 업로드해 주세요.

아래의 사용자 인풋를 받아 정림의 사내 워크스테이션으로 값을 json으로 보낸뒤
그래스호퍼를 실행, 결과값을 불러옵니다:

  1. 분석할 건물 혹은 파사드의 OBJ 파일
  2. 분석할 주변 건물 / 환경 OBJ 파일
  3. 기타 설정 값
